#  Scroll Car Animation 🚗💨

A modern, **responsive scroll animation** where a car moves along a road, leaving a trail, revealing letters, and showing animated statistic boxes. Built using **HTML, CSS, and GSAP**.

---

## LIVE 

(https://piyush-amola.github.io/Scroll-Driven-Hero-Section-Animation/))  


---

## Features

- **Smooth Car Animation:** The car moves horizontally across the road using GSAP.  
- **Floating Effect:** Car gently floats up and down for a realistic look.  
- **Dynamic Letters:** Letters appear sequentially as the car passes them.  
- **Trail Animation:** A colorful trail grows behind the car dynamically.  
- **Stat Boxes:** Glass-style stat boxes appear at specific positions along the road.  
- **Fully Responsive:** Works on desktop, tablet, and mobile with scaling car, road, letters, and boxes.  
- **Clean Design:** Modern neon color scheme with glassy stat cards.

---

## Technologies Used

- **HTML5** – Structure and semantic layout  
- **CSS3** – Styling, responsive design, glass effect, gradients  
- **GSAP (GreenSock Animation Platform)** – Smooth animations and floating effects  
- **JavaScript** – Dynamic updates of letters, trail, and boxes
